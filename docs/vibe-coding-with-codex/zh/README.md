# 中文教程目录

这个文件夹是一套完整的 `employee-map` 案例教程。  
目标不是讲感受，也不是讲一次聊天怎么被纠偏，而是手把手拆解：怎么把一组组织分析设计图，真正做成一个可运行、可导入、可下钻、可导出的前端产品原型。

## 适合谁看

这套教程适合这几类读者：

- 想把设计图做成真实前端产品的人
- 想用 Codex 做完整案例，而不是只生成页面的人
- 想练习“数据模型 + 导入 + 规则 + 导出”这类产品实现的人
- 想把项目整理成可公开学习 repo 的人

## 你读完以后应该得到什么

读完整套教程以后，你应该不只是理解这个 repo 做了什么，而是能自己复用这条路线去做下一类产品：

- 从设计图出发
- 拆能力
- 立数据模型
- 生成样例数据
- 做导入和规则
- 做测试
- 整理成教程型 repo
